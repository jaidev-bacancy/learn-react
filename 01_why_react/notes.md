# What I learnt?

### Benefits of using React:

 1. **Declarative:** When developing web apps in vanilla JS we need to refactor every part which may contain our updated variables (Imperative style), but React handles such scenarios by itself. Hence declarative.
 2. **Component-based:** React allows to create components (some HTML code), which can be reused again.
 3. **SPA (Single Page Application):** We can create SPAs using React which has the following  advantage: Once the page is loaded, moving to different pages (routes) within the application, does not refresh the entire page. All code is loaded into the browser when the application is initially loaded.
