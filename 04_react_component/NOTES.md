# What I learnt?
- **What is a Component? :** It is a **reusable** piece of UI. In technical terms it is a **function** which **returns** some **JSX**.
- **How React handles *Component* internally:** Internally while using a Component, the `type` attribute of React element is the **Component name** itself. Another notable thing is the **behaviour of props,** when the `type` of React element is **tag name** then the props are added as **attributes of that tag.** 
- Learnt how to pass **props** to **components**.
